## Обзор проекта

Генератор документов - это веб-приложение для создания официальных документов (указов и постановлений) с возможностью предпросмотра и скачивания в формате PNG.

## Структура файлов

Проект состоит из одного HTML-файла, содержащего:
- HTML-разметку
- CSS-стили  
- JavaScript-логику

## Как использовать

### Заполнение форм

1. **Выберите тип документа** через вкладки "Указы" или "Постановления"
2. **Заполните все необходимые поля**:
   - Номер документа
   - Дата
   - Должность и ФИО сотрудника
   - Должность и ФИО подписанта
   - Дополнительные параметры

3. **Нажмите "Сгенерировать"** для предпросмотра
4. **Нажмите "Скачать PNG"** для сохранения документа

### Настройки документов

#### Для указов:
- Выбор типа указа (назначение/увольнение)
- Включение/отключение водяного знака "ЧЕРНОВИК"

#### Для постановлений:
- Выбор отдела (PSID/LSFD)
- Редактирование графика деятельности
- Включение/отключение водяного знака "ЧЕРНОВИК"

## Настройка и кастомизация

### Изменение стилей

Все стили находятся в теге `<style>` в разделе `<head>`. Основные настраиваемые параметры:

```css
:root {
  --primary-color: #2c3e50;         /* Основной цвет текста */
  --secondary-color: #3498db;       /* Акцентный цвет */
  --background-gradient: ...        /* Градиент фона */
  /* и другие переменные */
}
```

### Изменение логотипов и печатей

Замените пути к файлам печатей в CSS:
```css
.stamp-placeholder {
  background-image: url('stamp.png'); /* Замените на ваш путь */
}

.stamp2-placeholder {
  background-image: url('stamp2.png'); /* Замените на ваш путь */
}
```

### Добавление новых типов документов

1. Добавьте новую вкладку в раздел с вкладками
2. Создайте соответствующую форму в разделе tab-content
3. Добавьте шаблон документа в раздел предпросмотра
4. Реализуйте логику генерации в JavaScript

## Развертывание на Netlify

### Шаг 1: Подготовка файлов

Убедитесь, что у вас есть:
- `index.html` (главный файл)
- `stamp.png` (изображение печати 1)
- `stamp2.png` (изображение печати 2)

### Шаг 2: Загрузка на Netlify

1. **Перейдите на [netlify.com](https://www.netlify.com/)**
2. **Зарегистрируйтесь** или войдите в аккаунт
3. **Нажмите "Add new site"** → "Deploy manually"
4. **Перетащите файлы** в область загрузки или выберите через проводник
5. **Дождитесь завершения** деплоя

### Шаг 3: Настройка домена (опционально)

1. В панели управления сайтом перейдите в **Site settings**
2. Выберите **Domain management**
3. Нажмите **Add custom domain** для подключения своего домена

### Шаг 4: Настройка редиректов (опционально)

Создайте файл `_redirects` в корне проекта для обработки SPA-маршрутизации:
```
/*    /index.html   200
```

## Особенности работы

### Темная тема
- Приложение поддерживает светлую и темную тему
- Настройка сохраняется в localStorage браузера
- Переключение через кнопку в правом верхнем углу

### Адаптивность
- Интерфейс адаптируется под мобильные устройства
- Предпросмотр документа масштабируется на маленьких экранах

### Генерация изображений
- Используется библиотека html2canvas для преобразования HTML в PNG
- Изображения генерируются с высоким качеством (scale: 3)

## Возможные проблемы и решения

### Проблема: Печати не отображаются
**Решение:** Убедитесь, что файлы `stamp.png` и `stamp2.png` находятся в корневой директории проекта

### Проблема: Ошибка при генерации PNG
**Решение:** 
- Загрузите сайт через веб-сервер (не открывайте файл напрямую)
- Проверьте консоль браузера на наличие ошибок CORS

### Проблема: Неправильные шрифты
**Решение:** Убедитесь в стабильном интернет-соединении (шрифты загружаются с Google Fonts)
